Descripción de la materia

Este curso sirve de introducción al desarrollo web y se enfoca en las tres tecnologías fundamentales de cualquier aplicación web: HTML, CSS y JavaScript. De esta manera, los estudiantes aprendemos a construir sitios web atractivos y funcionales. Además, se introduce JS para manejar contenido dinámico y eventos , se exploran frameworks como Bootstrap e incluso se incluye una introducción a la interacción con el backend mediante el almacenamiento de datos en Firebase.

Contenidos

  • Unidad 1: Introducción al HTML
  • Unidad 2: Introducción a las hojas de estilo en cascada (CSS)
  • Unidad 3: Maquetación de sitios web
  • Unidad 4: Frameworks de diseño web responsivo
  • Unidad 5: Introducción a JavaScript
  • Unidad 6: Modelo de objetos del documento (DOM) y JavaScript asíncrono
